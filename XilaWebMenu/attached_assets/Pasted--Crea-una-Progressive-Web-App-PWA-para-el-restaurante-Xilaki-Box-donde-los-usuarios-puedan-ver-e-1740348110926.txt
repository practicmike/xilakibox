2️⃣ Carrito de compras y pedido por WhatsApp:
Los usuarios pueden agregar múltiples productos antes de enviarlos.
Mostrar un resumen del pedido con el precio total antes de confirmar.
Generar un mensaje de WhatsApp Web con los detalles del pedido y abrir el chat del restaurante.
3️⃣ Admin Panel (sin autenticación por ahora):
Acceso sin credenciales, cualquier usuario puede gestionar el menú.
Funciones:
Agregar, editar y eliminar categorías (ej. Salsas, Proteínas, Toppings).
Agregar, editar y eliminar productos dentro de cada categoría.
Modificar nombres, descripciones, precios e imágenes.
Los cambios en el panel deben actualizar el menú en tiempo real.
Accesible desde una URL separada o un botón oculto en la app.
4️⃣ Tecnología recomendada:
Frontend: React + TailwindCSS para una interfaz moderna y responsiva.
Backend: Firebase Firestore (base de datos en tiempo real) o una API en Node.js con Express y MongoDB.
Estado global: Redux o Context API para manejar el carrito.
PWA: Soporte offline y opción de instalación en móviles.
